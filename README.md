# jQuery Mobile+blaze+meteor开发基于微信的H5小游戏
# 描述
> 基于微信的理财产品。
## 主要技术栈：
- jQuery Mobile
- blaze
- jQuery
- nodejs
- meteor

# 如何运行
## 下载项目
    git clone https://github.com/hszheng/runningTurkey.git
## 安装node环境
    curl https://install.meteor.com/ | sh
## 初始化项目
    meteor npm install
## 启动
    meteor --port 3006
> 以chrome为例，打开控制台并切换到手机模拟观看效果如下。
